A furniture suite of remixed parts.

The BA System uses minimal parts - legs, braces and two slabs - to make a suite of matching pieces. An exercise in restraint, each piece follows the same set of strict rules, giving the collection a strong sense of identity.

The kit-of-parts approach allows fluid adjustment if you need - combine a taller set of legs with a longer set of braces, fit a shelf between two console tables, extend a desk from the bookcase, whatever you can think of.
